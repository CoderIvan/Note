# Position

* Static
  * 默认
* Relative
  * 未脱离文档流
  * 相对原文档流位置定位
* Absolute
  * 脱离文档流
  * 相对最近非`static`祖先定位
* Fixed
  * 脱离文档流
  * 相对于viewport视口定位
* Sticky
  * relative + fixed
